# MuJing 幕境｜语境化开源背单词工具

死记硬背背单词，遗忘速度极快。
把单词放回**真实生活语境**中学习，才能真正记住、并且灵活使用。

开源项目 **MuJing（幕境）**，将电影、美剧、字幕、文档中的真实口语表达，打造成沉浸式词汇课堂：
在场景中理解含义，在重复出现中巩固记忆，通过专项练习彻底锁定单词。

- GitHub：[github.com/tangshimin/MuJing](https://github.com/tangshimin/MuJing)
- 在线地址：[mujingx.com](https://mujingx.com)

## ✨ 核心功能
- 🎬 **情境化记忆**：单词搭配原生影视例句，理解更容易、记忆更持久
- 💬 **智能弹幕提示**：观影自动高亮生词，边看边复习不费力
- 📝 **全套互动练习**：拼写、抄写、听写一体化训练
- 📚 **自定义词库**：MKV、字幕、文档一键导入生成个人词库
- ▶️ **智能字幕播放器**：字幕跳转、实时翻译，不打断观影节奏
- 🔗 **词库智能整合**：多来源内容自动关联，学习体系化

支持 Windows、macOS，提供现成安装包开箱即用。
适合想要**高效、长效、会用英语**的学习者。
