欢迎来到我的博客。

建这个博客的初衷很简单——我需要一个属于自己的数字花园，用来沉淀那些在碎片化阅读和日常开发中产生的想法。

## 为什么选择写博客？

在信息爆炸的时代，写博客看似是一种"古老"的方式。社交媒体上的短内容更容易获得即时反馈，但长文写作有着不可替代的价值：

- **强迫思考**：当你试图向别人解释一个概念时，你必须先把它彻底理解
- **建立索引**：大脑的存储是有限的，写下来是最好的外部记忆
- **连接他人**：你的困惑可能正是别人的困惑，你的解决方案可能正是别人需要的

## 技术选型

这个博客使用 [Astro](https://astro.build/) 构建，部署在 [Cloudflare Pages](https://pages.cloudflare.com/) 上。选择这套技术栈的原因：

1. **Astro** — 零 JavaScript 的默认输出，加载速度极快，对博客类内容站点来说是最优解
2. **Cloudflare Pages** — 全球 CDN，免费额度充足，从 GitHub 自动部署
3. **Tailwind CSS** — 原子化 CSS，快速开发且高度可定制

## 接下来写什么？

计划围绕以下几个方向更新内容：

- **技术实践**：前后端开发中的真实经验和踩坑记录
- **工具推荐**：提高效率的开发工具和工作流
- **读书笔记**：技术书籍和非虚构作品的阅读摘要
- **独立思考**：对行业趋势和个人成长的观察

希望你能在这里找到有价值的内容。如果有什么想交流的，欢迎通过 [GitHub](https://github.com/teloei) 联系我。
